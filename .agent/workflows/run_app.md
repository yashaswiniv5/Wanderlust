---
description: Run the frontend and backend of the application
---

1. Start the Backend Server
   - Open a new terminal.
   - Navigate to the server directory: `cd server`
   - Install dependencies if needed: `npm install`
   - Start the server:
// turbo
```bash
npm run dev
```

2. Start the Frontend Client
   - Open a new terminal.
   - Navigate to the client directory: `cd client`
   - Install dependencies if needed: `npm install`
   - Start the development server:
// turbo
```bash
npm run dev
```

3. Access the Application
   - Open your browser and navigate to `http://localhost:5173` (or the port shown in the client terminal).
